## Features

- Hash Texts/Strings
- Hash Files (Developing State)
- Check Hashes For Intergrity Of Files (Developing State)
- Algorithms Available
  - MD5
  - SHA-1
  - SHA-224
  - SHA-256
  - SHA-384
  - SHA-512
